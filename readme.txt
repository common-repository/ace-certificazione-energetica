Il plugin indispensabile per gestire la Certificazione Energetica ( ACE ) degli immobili.

== Description ==

Certificazione Energetica plugin, il plugin indispensabile per un sito di agenzia immobiliare o portale di annunci immobiliari.
CARATTERISTICHE:
- aggiunge ad ogni articolo (annuncio immobiliare) i dati della certificazione energetica.
- aggiunge un metabox in cui inserire i dati della certificazione energetica di ogni annuncio.
- aggiunge una colonna di riepilogo in edit.php per visualizzare più rapidamente gli annunci correttamente impostati.

== Installation ==

1. Unpack file package using your favorite zip software.
1. Upload the `ace-certificazione-energetica` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
1. Login to wordpress admin and go to plugin tab and activate Certificazione Energetica Plugin.
1. Go to Option => Certificazione Energetica tab to config the plugin option.

== Upgrade ==

1. Upgrade Certificazione Energetica is easy
1. Use the plugin uploader or
1. Replace plugin file (CertificazioneEnergetica.php) 
1. in plugin directory with new version. 
1. No need to re-config any options in “Certificazione Energetica” option tab.

== Configuration Panel ==

You can config the default appearance of your ACE window generate by Certificazione Energetica Plugin in the option tab in your wp-admin (wp-admin => option => Certificazione Energetica).
